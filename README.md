# NicePic

一个优雅的截图美化工具，为你的截图添加渐变背景、阴影和噪点纹理。

## 🚀 在线体验

**体验地址：** https://lessnoise.github.io/nicepic/

## 💡 项目说明

本项目由 **100% Vibe coding** 制作，使用以下工具开发：
- **AI Studio** - AI 辅助开发
- **Cursor** - AI 代码编辑器

## 🛠️ 本地运行

**前置要求：** Node.js

1. 安装依赖：
   ```bash
   npm install
   ```

2. 运行开发服务器：
   ```bash
   npm run dev
   ```

3. 构建生产版本：
   ```bash
   npm run build
   ```

## 📝 功能特性

- 🎨 多种渐变背景和纯色背景
- 🖼️ 实时预览效果
- 📐 可调节的阴影和圆角
- ✨ 噪点纹理效果
- 📱 响应式设计，支持移动端
- 🌐 中英文双语支持
